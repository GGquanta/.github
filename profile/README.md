# 国光量子 · 人工智能研究小组

国光量子人工智能研究小组聚焦 **量子科技与人工智能（Artificial Intelligence，AI）的交叉融合**，面向科研辅助、知识工程与垂直场景智能体，开展可复现的工程化研究与数据集建设。

---

## 研究方向

| 方向 | 概要 |
| --- | --- |
| 科研文献智能体 | 在规模化学术文献访问能力之上，构建多角色智能体工作流，覆盖检索、规划、证据综合、引用核查与分级受众表达。 |
| 量子计算实验辅助 | 将大语言模型与量子计算知识、混合编程及实验全链路（设计—提交—分析）相结合，降低量子科研与教学门槛。 |
| 科普与 STEM 数据工程 | 建设面向科普和科研的监督微调（Supervised Fine-Tuning，SFT）数据体系，强调封闭上下文、事实校订与可审计血缘。 |
| 文献知识结构化 | 从论文正文到实体词典、文献—实体映射与知识图谱扩展，支撑领域术语一致性与检索增强生成（Retrieval-Augmented Generation，RAG）。 |
| 量子安全与 AI 协同 | 在后量子密码（Post-Quantum Cryptography，PQC）、量子密钥分发（Quantum Key Distribution，QKD）等议题上，探索监测、测评与智能体安全通道等工程化路径（研究性表述，非产品承诺）。 |

我们持续关注 **「AI 推动量子认知与科研效率」** 与 **「量子能力增强可信计算与通信」** 的双向协同。

---

## 技术能力

- **多智能体编排**：Router、Planner、Coordinator、Researcher、Evaluator 等角色分工，强调步骤可追溯、证据链可导出。
- **混合检索增强**：向量检索与引文图、知识图谱（GraphRAG）动态组合，支持增量索引与冲突消解。
- **学术诚信能力**：多源文献交叉验证、误引检测与「主张—原文」一致性核对，降低长文本生成中的幻觉风险。
- **多模态文献理解**：面向图表、公式、表格等非文本内容的解析与联合表征。
- **工具化与协议**：文献接口封装为标准化工具（含模型上下文协议 Model Context Protocol，MCP 等），支持批处理、缓存与速率治理。
- **数据与评测闭环**：JSON Schema 校验、分组隔离评测集、精确与相似去重、三级质检轨道（文献驱动 / 蒸馏 / 事实裁决）及 registry 血缘。
- **领域知识底座**：量子通信、量子计算等方向的文献池、实体词典与 Markdown 知识管线，可横向扩展至能源、信息等 STEM 子域。
- **工程协作**：以 Git 工作流、Issues / Projects、文档即代码及 GitHub Actions 自动化流水线为主干。

## 开源与协作
